Copie de worldle, qu'on peut jouer en anglais, francais, espagnol, italien, allemand, portugais, neerlandais, polonais, suedois et norvégien
On peut également choisir la longueur du mot qu'on veut deviner et le nombre d'essais auxquels on a le droit
Le jeu se joue directement sur le terminal
Pour jouer, rentrer la fonction wordel(longueur du mot,nombre d'essais,langue)
